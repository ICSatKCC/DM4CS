Definition 11.1.8. Inverse Property.

Let \(\bullet \) be a binary operation on a set \(S\text{.}\) We say that \(\bullet \) has the inverse property if and only if for each \(a \in S\text{,}\) there exists \(b \in S\) such that \(a\bullet b = b\bullet a = e\text{.}\) We call \(b\) an inverse of \(a\text{.}\)
