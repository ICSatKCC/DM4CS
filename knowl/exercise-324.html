Exercise 5.

Write a complete trace of bubble sort on the list [6, 2, 3, 1, 5, 4] showing the lists obtained at each step (each change in values of \(i \) and \(j\)).
Answer.
1. [6 2 3 1 5 4] Before anything happens
2. [2 6 3 1 5 4] \(6>2\text{,}\) so swap
3. [2 3 6 1 5 4] \(6>3\text{,}\) so swap
4. [2 3 1 6 5 4] \(6>1\text{,}\) so swap
5. [2 3 1 5 6 4] \(6>5\text{,}\) so swap
6. [2 3 1 5 4 6] \(6>4\text{,}\) so swap
7. [2 1 3 5 4 6] \(2<3\text{,}\) so skip. \(3>1\text{,}\) so swap.
8. [2 1 3 4 5 6] \(3<5\text{,}\) so skip. \(5>4\text{,}\) so swap.
9. [1 2 3 4 5 6] \(5<6\text{,}\) so skip. \(1<2\text{,}\) so swap.
