procedure RecursiveMaximumElement:.
(\(\{a_1, a_2, ..., a_n\}\text{:}\) list of items; \(loc\text{:}\) location, starts at 1; \(max\text{:}\) largest element seen)
if \(loc > n\text{:}\)
return: max
else:
if loc = 1
then max := \(a_{1} \)
else if max \(\leq a_{loc}\)
then max := \(a_{loc} \)
return RecursiveMaximumElement(\(x\text{;}\) \(\{a_1, a_2, ..., a_n\}\text{;}\) \(loc+1\text{;}\) max)
