Exercise 1.

Find the point \((x, y )\) which lies on both lines, \(x + 3y = 1\) and \(4x - y = 3\text{.}\)
Answer.
Set up augmented matrix:
\begin{equation*} \left( \begin{array}{ccc} 1 \amp 3 \amp 1\\ 4 \amp -1 \amp 3\\ \end{array} \right). \end{equation*}
Add \(-4 *\) the first line to the second.
\begin{equation*} \left( \begin{array}{ccc} 1 \amp 3 \amp 1\\ 0 \amp -13 \amp -1\\ \end{array} \right). \end{equation*}
Divide line two by \(-13\)
\begin{equation*} \left( \begin{array}{ccc} 1 \amp 3 \amp 1\\ 0 \amp 1 \amp 1/13\\ \end{array} \right). \end{equation*}
So \(y = 1/13\text{.}\) Backsubstituting: \(x + 3(1/13) = 1\text{,}\) so \(x = 10/13\text{.}\) Therefore \((x, y) \) is the point \((10/13,1/13)\text{.}\)
