Set up augmented matrix:
\begin{equation*} \left( \begin{array}{ccc} 1 \amp 2 \amp 1\\ 2 \amp -1 \amp 1\\ 4 \amp 3 \amp 3\\ \end{array} \right). \end{equation*}
Add \(-4 *\) the first line to the third.
\begin{equation*} \left( \begin{array}{ccc} 1 \amp 2 \amp 1\\ 2 \amp -1 \amp 1\\ 0 \amp -5 \amp -1\\ \end{array} \right). \end{equation*}
Add \(-2 *\) the first line to the second.
\begin{equation*} \left( \begin{array}{ccc} 1 \amp 2 \amp 1\\ 0 \amp -5 \amp -1\\ 0 \amp -5 \amp -1\\ \end{array} \right). \end{equation*}
Add \(-1 *\) the second line to the third.
\begin{equation*} \left( \begin{array}{ccc} 1 \amp 2 \amp 1\\ 0 \amp -5 \amp -1\\ 0 \amp 0 \amp 0\\ \end{array} \right). \end{equation*}
Divide line two by \(-5\)
\begin{equation*} \left( \begin{array}{ccc} 1 \amp 2 \amp 1\\ 0 \amp 1 \amp 1/5\\ 0 \amp 0 \amp 0\\ \end{array} \right). \end{equation*}
So \(y = 1/5\text{.}\) Backsubstituting to line one: \(x + 2(1/5) = 1\text{,}\) so \(x = 3/5\text{.}\) The common intersecting point is \((3/5, 1/5)\text{.}\)
