A more complete solution can be obtained if we represent \(n\) in binary form. For each \(n\geq 1\text{,}\) there exists a non-negative integer \(r\) such that
\begin{gather} \quad \quad 2^{r-1}\leq n < 2^r\tag{13.2.3} \end{gather}
For example, if \(n = 21\text{,}\) \(2^4 \leq 21 < 2^5\text{;}\) therefore, \(r = 5\text{.}\) If \(n\) satisfies \(13.2.3), its binary representation requires \(r\) digits. For example, \(21_{\text{ten}}\) = \(10101_{\textrm{two}}\text{.}\)
