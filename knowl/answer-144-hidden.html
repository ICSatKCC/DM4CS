LargestAndSmallest.
(\(\{a_1, a_2, ..., a_n\}\text{:}\) list of integers)
\(smallest := a_1\)
\(largest := a_1\)
\(index:= 2\)
while \(index \leq n\text{:}\)
if \(smallest \geq a_{index}\text{:}\)
\(smallest := a_{index}\)
if \(largest \leq a_{index}\text{:}\)
\(largest := a_{index}\)
\(index := index + 1\)
return \(smallest + largest\)
