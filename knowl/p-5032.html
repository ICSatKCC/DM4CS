Base case: Suppose \(w\) has length 0. The only string \(w\) with length 0 is \(\varep\text{,}\) so we want to show that \(\delta^*(q_0,\varep)\) corresponds to \(\partial^*(p_0,\varep)\text{.}\) Well, \(\delta^*(q_0, \varep) = q_0\text{,}\) since in a DFA, \(\delta^*(q, \varep) = q\) for any state~ \(q\text{.}\) We explicitly made \(q_0\) correspond to \(\partial^*(p_0,\varep)\text{,}\) and so the property holds for \(w\) with length 0.
