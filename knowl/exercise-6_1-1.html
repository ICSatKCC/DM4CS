Exercise 1.

Prove the following:
  1. Let \(A\text{,}\) \(B\text{,}\) and \(C\) be sets \(\neq \emptyset\text{.}\) If \(A\subseteq B\) and \(B\subseteq C\text{,}\) then \(A\subseteq C\text{.}\)
  2. Let \(A\) and \(B\) be sets \(\neq \emptyset\text{.}\) Then \(A - B= A\cap \overline{B}\) .
  3. Let \(A,B, \textrm{ and } C\) be sets \(\neq \emptyset\text{.}\) If (\(A\subseteq B\) and \(A\subseteq C\)) then \(A\subseteq B\cap C\text{.}\)
  4. Let \(A \textrm{ and } B\) be sets \(\neq \emptyset\text{.}\) \(A\subseteq B\) if and only if \(\overline{B}\subseteq \overline{A}\) .
  5. Let \(A,B, \textrm{ and } C\) be sets \(\neq \emptyset\text{.}\) If \(A\subseteq B\) then \(A\times C \subseteq B\times C\text{.}\)
