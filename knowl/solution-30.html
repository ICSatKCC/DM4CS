Solution 11.1.18.1.

The augmented matrix is
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp -1 \amp 1 \amp 3\\ 1 \amp 1 \amp -1 \amp 1 \amp 1\\ 1 \amp 3 \amp -1 \amp 1 \amp 5\\ \end{array} \right) \end{equation*}
Take -1 times the first row and add to the second. Then take -1 times the first row and add to the third. This yields
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp -1 \amp 1 \amp 3\\ 0 \amp -1 \amp 0 \amp 0 \amp -2\\ 0 \amp 1 \amp 0 \amp 0 \amp 2\\ \end{array} \right) \end{equation*}
Now add the second row to the bottom row
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp -1 \amp 1 \amp 3\\ 0 \amp -1 \amp 0 \amp 0 \amp -2\\ 0 \amp 0 \amp 0 \amp 0 \amp 0\\ \end{array} \right) \end{equation*}
This matrix is in echelon form and you see the basic variables are \(x\) and \(y\) while the free variables are \(z\) and \(w\text{.}\) Assign \(s\) to \(z\) and \(t\) to \(w\text{.}\) Then the second row yields the equation, \(y = 2\) while the top equation yields the equation, \(x + 2y - s + t = 3\) and so since \(y = 2\text{,}\) this gives \(x + 4 - s + t = 3\) showing that \(x = -1 + s - t\text{,}\) \(y = 2\text{,}\) \(z = s\text{,}\) and \(w = t\text{.}\) One can write this in the form
\begin{equation*} \left( \begin{array}{c} x\\ y\\ z\\ w\\ \end{array} \right) = \left( \begin{array}{c} -1 + s - t\\ 2\\ s\\ t\\ \end{array} \right) \end{equation*}
This is another example of a system which has an infinite solution set but this time the solution set depends on two parameters, not one.
Most people find it less confusing in the case of an infinite solution set to first place the augmented matrix in row reduced echelon form rather than just echelon form before seeking to write down the description of the solution. In the above, this means we donâ€™t stop with the echelon form above. Instead we first place it in reduced echelon form as follows.
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 0 \amp -1 \amp 1 \amp -1\\ 0 \amp 1 \amp 0 \amp 0 \amp 2\\ 0 \amp 0 \amp 0 \amp 0 \amp 0\\ \end{array} \right) \end{equation*}
Then the solution is \(y = 2\) from the second row and \(x = -1 + z - w\) from the first. Thus letting \(z = s\) and \(w = t\text{,}\) the solution as is given previously.
