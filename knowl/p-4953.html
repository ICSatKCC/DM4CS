Give regular expressions over \(\Sigma=\ab\) that generate the following languages.
  1. \(\displaystyle L_1 = \{ x \ | \ x \mbox{ contains 3 consecutive $a$'s}\}\)
  2. \(\displaystyle L_2 = \{ x \ | \ x \mbox{ has even length}\}\)
  3. \(\displaystyle L_3 = \{ x \ | \ n_b(x) = 2 \bmod{3}\}\)
  4. \(\displaystyle L_4 = \{ x \ | \ x \mbox{ contains the substring } aaba\}\)
  5. \(\displaystyle L_5 = \{ x \ | \ n_b(x) \lt 2 \}\)
  6. \(\displaystyle L_6 = \{ x \ | \ x \mbox{ doesn't end in } aa\}\)
