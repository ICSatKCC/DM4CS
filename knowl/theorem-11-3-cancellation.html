Theorem 11.3.8. Cancellation Laws.

If \(a\text{,}\) \(b\text{,}\) and \(c\) are elements of group \(G\text{,}\) then
\begin{equation*} \begin{array}{lc} \textrm{left cancellation:}& (a \bullet b = a \bullet c) \Rightarrow b = c\\ \textrm{right cancellation:}& (b \bullet a = c \bullet a) \Rightarrow b = c\\ \end{array} \end{equation*}
