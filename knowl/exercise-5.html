Exercise 5.

Let \(A = \{0, 2, 3\}\text{,}\) \(B = \{2, 3\}\text{,}\) and \(C = \{1, 5, 9\}\text{.}\) Determine which of the following statements are true. Give reasons for your answers.
  1. \(\displaystyle 3 \in A\)
  2. \(\displaystyle \{3\} \in A\)
  3. \(\displaystyle \{3\} \subseteq A\)
  4. \(\displaystyle B\subseteq A\)
  5. \(\displaystyle A\subseteq B\)
  6. \(\displaystyle \emptyset \subseteq C\)
  7. \(\displaystyle \emptyset \in A\)
  8. \(\displaystyle A\subseteq A\)
Answer.
  1. True
  2. False
  3. True
  4. True
  5. False
  6. True
  7. False
  8. True
