Algorithm 9.2.3. Recursive Linear Search.
procedure recursive_linear_search:.
(\(x\text{:}\) target; \(\{a_1, a_2, ..., a_n\}\text{:}\) list of items; \(loc\text{:}\) location, starts at 1)
If \(loc > n\text{:}\)
return: 0
Else If\(x = a_{loc}\text{:}\)
return: loc
Else:
return recursive_linear_search(\(x\text{;}\) \(\{a_1, a_2, ..., a_n\}\text{;}\) \(loc+1\))
