Exercise 15.

Give the complete solution to the system of equations, \(7x + 14y + 15z = 22\) , \(2x + 4y + 3z = 5\text{,}\) and \(3x + 6y + 10z = 13\text{.}\)
Solution.
We have the corresponding augmented matrix:
\begin{equation*} \left( \begin{array}{ccc|c} 7 \amp 14 \amp 15 \amp 22\\ 2 \amp 4 \amp 3 \amp 5\\ 3 \amp 6 \amp 10 \amp 13\\ \end{array} \right). \end{equation*}
Add \(-2 * \) row one to \(7 * \) row 2:
\begin{equation*} \left( \begin{array}{ccc|c} 7 \amp 14 \amp 15 \amp 22\\ 0 \amp 0 \amp -9 \amp -9\\ 3 \amp 6 \amp 10 \amp 13\\ \end{array} \right). \end{equation*}
Add \(-3 * \) row one to \(7 * \) row 3:
\begin{equation*} \left( \begin{array}{ccc|c} 7 \amp 14 \amp 15 \amp 22\\ 0 \amp 0 \amp -9 \amp -9\\ 0 \amp 0 \amp 25 \amp 25\\ \end{array} \right). \end{equation*}
Multiply row 2 by \(-1/9\) and row 3 by \(1/25\)
\begin{equation*} \left( \begin{array}{ccc|c} 7 \amp 14 \amp 15 \amp 22\\ 0 \amp 0 \amp 1 \amp 1\\ 0 \amp 0 \amp 1 \amp 1\\ \end{array} \right). \end{equation*}
So.... \(z = 1\text{.}\) Add \(-1 * \) row 2 to row 3:
\begin{equation*} \left( \begin{array}{ccc|c} 7 \amp 14 \amp 15 \amp 22\\ 0 \amp 0 \amp 1 \amp 1\\ 0 \amp 0 \amp 0 \amp 0\\ \end{array} \right). \end{equation*}
This shows that \(y\) is going to be a free variable. Assign \(s\) to \(y\text{.}\)The top row yields the equation \(7x + 14s + 15z = 22\text{,}\) since \(z = 1\text{,}\) this gives \(7x + 14s + 15 = 22\text{.}\) Solving for \(x\) we get \(x = 1 - 2s \text{,}\) so the final solution can be written:
\begin{equation*} \left( \begin{array}{c} x\\ y\\ z\\ \end{array} \right) = \left( \begin{array}{c} 1 - 2s\\ s\\ 1\\ \end{array} \right) \end{equation*}
