Definition 11.2.4. Group.

A group consists of a nonempty set \(G\) and a binary operation \(\bullet \) on \(G\) satisfying the properties
  1. \(\bullet \) is associative on \(G\text{:}\) \((a\bullet b)\bullet c=a\bullet (b\bullet c)\) for all \(a, b, c \in G\text{.}\)
  2. There exists an identity element, \(e \in G\text{,}\) such that \(a\bullet e=e\bullet a=a\) for all \(a \in G\text{.}\)
  3. For all \(a \in G\text{,}\) there exists an inverse; that is, there exist \(b\in G\) such that \(a \bullet b = b\bullet a=e\text{.}\)
