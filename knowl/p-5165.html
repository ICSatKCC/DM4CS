Suppose \(G\) is a context-free grammar with start symbol \(S\) and suppose \(w\in L(G)\text{.}\) By definition, this means that there is a sequence of one or more applications of production rules which produces the string \(w\) from \(S\text{.}\) This sequence has the form \(S\YIELDS x_1\YIELDS x_2\YIELDS\cdots\YIELDS w\text{.}\) Such a sequence is called a derivation of \(w\) (in the grammar \(G\)). Note that \(w\) might have more than one derivation. That is, it might be possible to produce \(w\) in several different ways.
