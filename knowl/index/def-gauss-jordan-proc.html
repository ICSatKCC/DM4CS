Definition 11.3.5. Gauss-Jordan Procedure.

Suppose \(A\) is an \(n \times n\) matrix. To find \(A^{-1}\) if it exists, form the augmented \(n \times 2n\) matrix
\begin{equation*} \left(A | I \right) \end{equation*}
and then, if possible do row operations until you obtain an \(n \times 2n\) matrix of the form
\begin{equation} \left(I|B\right).\tag{11.3.3} \end{equation}
When this has been done, \(B = A^{-1}\text{.}\) If it is impossible to row reduce to a matrix of the form \((I|B)\text{,}\) then \(A\) has no inverse.
