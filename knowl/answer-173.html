Answer 11.1.4.5.1.

  1. \(\bullet \) is commutative since \(\left| a-b\right| =\left| b-a\right|\) for all \(a, b \in \mathbb{N}\)
  2. \(\bullet \) is not associative. Take \(a = 1\text{,}\) \(b = 2\text{,}\) and \(c = 3\text{,}\) then \((a \bullet b) \bullet c =\left| \left| 1-2\right| -3\right| =2\) , and \(a \bullet (b \bullet c) = \left| 1-\left| 2-3\right| \right| = 0\text{.}\)
  3. Zero is the identity for \(\bullet \) on \(\mathbb{N}\text{,}\) since \(a\bullet 0=\left| a - 0\right| = a = \left| 0-a\right| = 0 \bullet a.\)
  4. Each element of \(\mathbb{N}\) inverts itself since \(a \bullet a=\left| a-a\right| = 0\text{.}\)
  5. \(\bullet \) is not idempotent, since, for \(a\neq 0\text{,}\) \(a \bullet a =0 \neq a\text{.}\)
