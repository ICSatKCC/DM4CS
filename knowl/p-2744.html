Algorithm 9.1.12. Iterative Binary Search.
procedure binary search.
(\(x\text{:}\) target; \(\{a_1, a_2, ..., a_n\}\text{:}\) sorted list of items)
loc := -1
begin := 1
end := n
found := FALSE
while (begin \(\leq\) end) and (not found)
mid := (begin + end) / 2
if \(x = a_{mid}\)
then loc := mid
found := TRUE
else if \(x < a_{mid}\)
then end := mid - 1
else
begin := mid + 1
return loc
