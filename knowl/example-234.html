Example 15.2.3.

Let \(\Sigma = \ab\text{,}\) and consider the regular expression \(r=a^*b^*\text{.}\) What is \(L(r)\text{?}\) Well, \(L(a) = \{a\}\) so \(L(a^*) = (L(a))^* = \{a\}^*\text{,}\) and \(\{a\}^*\) is the set of all strings of zero or more \(a\)’s, so \(L(a^*) = \{\varep, a, aa, aaa, \ldots\}\text{.}\) Similarly, \(L(b^*) = \{\varep, b, bb, bbb, \ldots\}\text{.}\) Since \(L(a^*b^*) = L(a^*)L(b^*) = \{xy \ | \ x\in L(a^*)\land y\in L(b^*)\}\text{,}\) we have \(L(a^*b^*) = \{\varep, a, b, aa, ab, bb, aaa, aab, abb, bbb, \ldots\}\text{,}\) which is the set of all strings of the form ``zero or more \(a\)’s followed by zero or more \(b\)’s".
