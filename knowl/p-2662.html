In addition to polynomials, we have the following Big O estimates:
  1. Any power of \(n\) is Big O any larger power of \(n\text{,}\) powers greater than 1:
    \begin{equation*} n^c = O(n^d), 1 < c < d. \end{equation*}
  2. A logarithm of \(n\) to a power is Big O any power of \(n\text{,}\) for log bases greater than 1:
    \begin{equation*} (log_b n)^c = O(n^d), 0 < c, d. \end{equation*}
  3. Any power of \(n\) is Big O an exponential in \(n\text{:}\)
    \begin{equation*} n^c = O(b^n), 1 < b, 0 < c. \end{equation*}
  4. Any exponential is Big O an exponential of a larger constant:
    \begin{equation*} b^n = O(c^n), 1 < b < c \end{equation*}
