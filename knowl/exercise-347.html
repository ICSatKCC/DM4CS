Exercise 4.

  1. Let \(A\) be the set of strings of 0’s and 1’s of length 3 or less. Define the relation of \(D\) on \(A\) by \(x D y\) if \(x\) is contained within \(y\text{.}\) For example, \(01 D 101\text{.}\) Draw a digraph for this relation.
  2. Do the same for the relation \(P\) defined by \(x P y\) if \(x\) is a prefix of \(y\text{.}\) For example, \(10 P 101\text{,}\) but \(01 P 101\) is false.
