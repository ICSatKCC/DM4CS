Item.
\(f \) is onto.
Proof: Let \(b \in 2\mathbb{P}\text{.}\) We want to show that there exists an element \(a \in \mathbb{P}\) such that \(f(a) = b\text{.}\) If \(b \in 2\mathbb{P}\text{,}\) \(b = 2k\) for some \(k \in \mathbb{P}\) by the definition of \(2\mathbb{P}\text{.}\) So we have \(f(k) = 2k = b\text{.}\) Hence, each element of 2\(\mathbb{P}\) is the image of some element of \(\mathbb{P}\text{.}\)
