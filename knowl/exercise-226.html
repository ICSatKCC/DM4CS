Exercise 5.

  1. Partition \(A = \{0, 1, 2, 3, 4, 5\}\) with the minsets generated by \(B_1= \{0, 2, 4\}\text{ }\)and \(B_2= \{1, 5\}\text{.}\)
  2. How many different subsets of \(A\) can you generate from \(B_1 \textrm{ and } B_2\text{?}\)
Answer.
  1. \(B_1\cap B_2=\emptyset\text{,}\) \(B_1\cap \overline{B_{2}}=\{0,2,4\}\text{,}\) \(\overline{B_{1}}\cap B_2=\{1,5\}\text{,}\) \(\overline{B_{1}}\cap \overline{B_{2}}=\{3\}\)
  2. \(2^3\text{,}\) since there are 3 nonempty minsets.
