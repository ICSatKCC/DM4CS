Theorem 17.6.6.

Let \(\Sigma\) be an alphabet and let \(L\) be a language over \(\Sigma\text{.}\) Then the following are equivalent:
  1. There is a Turing machine that accepts \(L\text{.}\)
  2. There is a two-tape Turing machine that runs forever, making a list of strings on its second tape, such that a string \(w\) is in the list if and only if \(w\in L\text{.}\)
  3. There is a Turing-computable function \(f\colon\{a\}^*\to\Sigma^*\) such that \(L\) is the range of the function \(f\text{.}\)
