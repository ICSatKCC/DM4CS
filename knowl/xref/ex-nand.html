Exercise 8.

The significance of NAND is that it is a “universal” operation in that all other logical operations can be built from it. It is shown in Table 4.3.15 that \(p \uparrow q \Leftrightarrow \neg (p \land q)\text{.}\)
  1. Prove that \(\neg p \Leftrightarrow p \uparrow p\text{.}\)
  2. Build \(\land\) using only \(\uparrow\text{.}\)
  3. Build \(\lor\) using only \(\uparrow\text{.}\)
