Example 15.1.11.

Let \(\Sigma = \{0,1\}\text{.}\) Then the following are all languages over \(\Sigma\text{:}\)
  • \(\displaystyle L_1 = \{011, 1010,111\}\)
  • \(\displaystyle L_2 = \{0, 10, 110, 1110, 11110, \ldots\}\)
  • \(L_3 = \{x \in \Sigma^* \ | \ n_0(x) = n_1(x) \}\text{,}\) where the notation \nw{\(n_0(x)\)} stands for the number of 0's in the string \(x\text{,}\) and similarly for \(n_1(x)\)
  • \(\displaystyle L_4 = \{x| x \mbox{ represents a multiple of 5 in binary}\}\)
