Exercise 3.

Describe the following sets using set-builder notation.
  1. \(\displaystyle \{ 5, 7, 9, \dots , 77, 79\}\)
  2. the rational numbers that are strictly between \(-1\) and \(1\)
  3. the even integers
  4. \(\displaystyle \{-18, -9,0,9, 18,27, \dots \}\)
Answer.
  1. \(\displaystyle \{2k + 1 \mid k \in \mathbb{Z}, 2 \leqslant k \leqslant 39\}\)
  2. \(\displaystyle \{x \in \mathbb{Q}\mid -1 < x < 1\}\)
  3. \(\displaystyle \{2n\mid n \in \mathbb{Z}\}\)
  4. \(\displaystyle \{9n\mid n \in \mathbb{Z}, -2 \leq n\}\)
