Algorithm 9.3.8. Selection Sort.
procedure SelectionSort.
(\(a_1, a_2, a_3, ... ,a_n\text{:}\) list of \(n\) numbers)
for \(j = 1\) to \(n-1\)
\(minLoc = j\)
for \(i = j+1\) to \(n\)
if \(a_i < a_{minLoc}\) then \(minLoc = i\)
if \(minLoc \neq j\)
swap \(a_j\) and \(a_{minLoc}\)
