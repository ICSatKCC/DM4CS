Next take -1 times the middle row and add to the bottom. Here the 1 in the second row is the pivot.
\begin{equation*} \left( \begin{array}{cccc} 3 \amp -1 \amp -5 \amp 9\\ 0 \amp 1 \amp -10 \amp 0\\ 0 \amp 0 \amp 0 \amp 0\\ \end{array} \right) \end{equation*}
This is in reduced echelon form. The equations corresponding to this reduced echelon form are \(y = 10z\) and \(x = 3 + 5z\text{.}\) Apparently \(z\) can equal any number. Lets call this number \(t\text{.}\)
In this context \(t\) is called a parameter.
Therefore, the solution set of this system is \(x = 3 + 5t\text{,}\) \(y = 10t\text{,}\) and \(z = t\) where \(t\) is completely arbitrary. The system has an infinite set of solutions which are given in the above simple way. This is what it is all about, finding the solutions to the system.
