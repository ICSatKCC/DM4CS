  1. We stress that the image of each ordered pair must be in \(S\text{.}\) This requirement disqualifies subtraction on the natural numbers from consideration as a binary operation, since \(1 - 2\) is not a natural number. Subtraction is a binary operation on the integers.
  2. On Notation. Despite the fact that a binary operation is a function, symbols, not letters, are used to name them. We will use a bullet point symbol, \(\bullet\) to represent an arbitrary binary operation. We will also use a diamond, \(\diamond\text{,}\) when a second symbol is needed. Another commonly used symbol is an asterisk, \(*\text{,}\) but that tends to be confusing due to its use as the multiplication operator in many programming languages so we will avoid it.
