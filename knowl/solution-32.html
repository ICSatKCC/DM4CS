Solution 11.1.3.9.1.

What we want is for one of the rows to be all zeros except for the last term. If we multiply the top row by \(-3/2\) and add that to the bottom row, we get
\begin{equation*} \left( \begin{array}{cc|c} 2 \amp h \amp 4\\ 0 \amp 6 +(-3/2 * h) \amp -1\\ \end{array} \right). \end{equation*}
We want to make an \(h\) such that the \(6 +(-3/2 * h) = 0\text{.}\) Solving for \(h\text{,}\) we get 4.
\begin{equation*} \left( \begin{array}{cc|c} 2 \amp 4 \amp 4\\ 0 \amp 6 +(-3/2 * 4) \amp -1\\ \end{array} \right) \end{equation*}
which simplifies to
\begin{equation*} \left( \begin{array}{cc|c} 2 \amp 4 \amp 4\\ 0 \amp 0 \amp -1\\ \end{array} \right)\text{,} \end{equation*}
which is inconsistent.
