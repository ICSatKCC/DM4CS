Exercise 1.

  1. Let \(B = \{a, b\}\) and \(U = \mathcal{P}(B)\text{.}\) Draw a Hasse diagram for \(\subseteq \) on \(U\text{.}\)
  2. Let \(A = \{1,2, 3, 6\}\text{.}\) Show that divides, \(\mid \text{,}\) is a partial ordering on \(A\text{.}\)
  3. Draw a Hasse diagram for divides on \(A\text{.}\)
  4. Compare the graphs of parts a and c.
Answer.
  1. See Figure 10.3.16.
  2. \(\mid\) on \(A\) is reflexive because \(\forall a \in A, a \mid a\) is true and \((a,a) \in \) the relation. \(\mid\) on \(A\) is antisymmetric because \(\forall a, b \in A\) if \(a \mid b\) is true and \((a,b) \in \) the relation, then \(b \mid a\) is not true and \((b,a) \notin \) the relation, unless \(a = b\text{.}\) \(\mid\) on \(A\) is transitive because \(\forall a, b, c \in A\text{,}\) if \((a,b)\) and \((b,c) \in\) the relation, then \(a|b\) and \(b|c\text{.}\) Therefore \(ak =b\) for some integer \(k\) and \(bj = c\) for some integer \(j\text{.}\) Thus \(akj = c\text{,}\) \(a|c\text{,}\) and \((a,c) \in\) the relation. Therefore \(\mid\) on \(A\) is a partial ordering.
  3. See Figure 10.3.16.
  4. The graphs are the same if we disregard the names of the vertices.
