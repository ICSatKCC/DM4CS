Definition 7.1.4. Equality of Matrices.

A matrix \(A\) is said to be equal to matrix \(B\) (written \(A = B\)) if and only if:
  1. \(A\) and \(B\) have the same order, and
  2. all corresponding entries are equal: that is, \(a_{ij}\) = \(b_{ij}\) for all appropriate \(i\) and \(j\text{.}\)
