Definition 11.1.10. Left Distributive Property.

Let \(\bullet \) and \(\diamond\) be binary operations on a set \(S\text{.}\) We say that \(\diamond\) is left distributive over \(\bullet\) if and only if \(a \diamond (b \bullet c) = (a \diamond b) \bullet (a \diamond c)\) for all \(a,b,c\in S\text{.}\)
