\(n! = n * (n-1) * \dots * 2 * 1\) for \(n \geq 1\text{.}\) There are \(n\) terms total. Since \(n \geq 1 \text{,}\) it must be true that:
\begin{equation*} n * (n-1) * \dots * 2 * 1 \leq n * n * \dots *n * n = n^n \end{equation*}
Therefore, \(n! = O(n^n)\) with \(c = 1\) and \(k = 1\text{.}\)
