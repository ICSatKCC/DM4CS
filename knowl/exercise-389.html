Exercise 11.

Determine if the system is consistent. If so, is the solution unique?
\begin{equation*} x + 2y + z - w = 2 \end{equation*}
\begin{equation*} x - y + z + w = 1 \end{equation*}
\begin{equation*} 2x + y -z = 1 \end{equation*}
\begin{equation*} 4x + 2y + z = 5 \end{equation*}
Solution.
We have the corresponding matrix:
\begin{equation*} \left( \begin{array}{cccc|c} 1 \amp 2 \amp 1 \amp -1 \amp 2 \\ 1 \amp -1 \amp 1 \amp 1 \amp 1\\ 2 \amp 1 \amp -1 \amp 0 \amp 1\\ 4 \amp 2 \amp 1 \amp 0 \amp 5\\ \end{array} \right). \end{equation*}
Add \(-4 * \) row 1 to row 4:
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp 1 \amp -1 \amp 2 \\ 1 \amp -1 \amp 1 \amp 1 \amp 1\\ 2 \amp 1 \amp -1 \amp 0 \amp 1\\ 0 \amp -6 \amp -3 \amp 4 \amp -3\\ \end{array} \right). \end{equation*}
Add \(-2 * \) row 1 to row 3:
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp 1 \amp -1 \amp 2 \\ 1 \amp -1 \amp 1 \amp 1 \amp 1\\ 0 \amp -3 \amp -3 \amp 2 \amp -3\\ 0 \amp -6 \amp -3 \amp 4 \amp -3\\ \end{array} \right). \end{equation*}
Add \(-1 * \) row 1 to row 2:
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp 1 \amp -1 \amp 2 \\ 0 \amp -3 \amp 0 \amp 2 \amp -1\\ 0 \amp -3 \amp -3 \amp 2 \amp -3\\ 0 \amp -6 \amp -3 \amp 4 \amp -3\\ \end{array} \right). \end{equation*}
Add \(-1 * \) row 2 to row 3:
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp 1 \amp -1 \amp 2 \\ 0 \amp -3 \amp 0 \amp 2 \amp -1\\ 0 \amp 0 \amp -3 \amp 0 \amp -2\\ 0 \amp -6 \amp -3 \amp 4 \amp -3\\ \end{array} \right). \end{equation*}
Add \(-2 * \) row 2 to row 4:
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp 1 \amp -1 \amp 2 \\ 0 \amp -3 \amp 0 \amp 2 \amp -1\\ 0 \amp 0 \amp -3 \amp 0 \amp -2\\ 0 \amp 0 \amp -3 \amp 0 \amp 1\\ \end{array} \right). \end{equation*}
Therefore we see that we have a problem because rows 3 and 4 are saying \(-3z= -2\) and \(-3z = 1\text{.}\) Or we could keep going with the row operations and add \(-1 * \)row 3 to row 4: Add \(-2 * \) row 2 to row 4:
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp 1 \amp -1 \amp 2 \\ 0 \amp -3 \amp 0 \amp 2 \amp -1\\ 0 \amp 0 \amp -3 \amp 0 \amp -2\\ 0 \amp 0 \amp 0 \amp 0 \amp 3\\ \end{array} \right) \end{equation*}
which then definitively shows that the system is not consistent.
