Exercise 4.

For each pair of functions below, determine whether \(f(x) = O(g(x))\) and whether \(g(x) = O(f(x))\text{.}\) If one function is Big O of the other, give values for \(c\) and \(k\text{.}\)
  1. \(\displaystyle f(x) = x^2, g(x) = 3x.\)
  2. \(\displaystyle f(x) = \frac{3x - 7}{x+4}, g(x) = 4.\)
  3. \(\displaystyle f(x) = 2^x, g(x) = 2^{\frac{x}{2}}.\)
  4. \(\displaystyle f(x) = \log (n!), g(x) = \log (n^n).\)
  5. \(\displaystyle f(x) = x^{10}, g(x) = 10^x.\)
