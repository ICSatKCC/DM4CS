Now add the second row to the bottom row
\begin{equation*} \left( \begin{array}{ccccc} 1 \amp 2 \amp -1 \amp 1 \amp 3\\ 0 \amp -1 \amp 0 \amp 0 \amp -2\\ 0 \amp 0 \amp 0 \amp 0 \amp 0\\ \end{array} \right) \end{equation*}
This matrix is in echelon form and you see the basic variables are \(x\) and \(y\) while the free variables are \(z\) and \(w\text{.}\) Assign \(s\) to \(z\) and \(t\) to \(w\text{.}\) Then the second row yields the equation, \(y = 2\) while the top equation yields the equation, \(x + 2y - s + t = 3\) and so since \(y = 2\text{,}\) this gives \(x + 4 - s + t = 3\) showing that \(x = -1 + s - t\text{,}\) \(y = 2\text{,}\) \(z = s\text{,}\) and \(w = t\text{.}\) One can write this in the form
\begin{equation*} \left( \begin{array}{c} x\\ y\\ z\\ w\\ \end{array} \right) = \left( \begin{array}{c} -1 + s - t\\ 2\\ s\\ t\\ \end{array} \right) \end{equation*}
This is another example of a system which has an infinite solution set but this time the solution set depends on two parameters, not one.
