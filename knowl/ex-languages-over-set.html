Example 15.1.11. Languages over a set.

Let \(\Sigma = \{0,1\}\text{.}\) Then the following are all languages over \(\Sigma\text{:}\)
\begin{align*} L_1 \amp = \{011, 1010,111\} \\ L_2 \amp = \{0, 10, 110, 1110, 11110, \ldots\} \\ L_3 \amp = \{x \in \Sigma^* \ | \ n_0(x) = n_1(x) \} \\ \amp\text{where the notation } n_0(x) \text{ stands for the number of 0's in the string } x \\ \amp \text{and similarly for } n_1(x)\\ L_4 \amp = \{x| x \mbox{ represents a multiple of 5 in binary}\} \end{align*}
